

# DeepScan-AI: Deepfake Image Detection System

##  Project Overview
DeepScan-AI is a full-stack digital forensic tool designed to combat the rising threat of AI-generated misinformation. Using advanced Deep Convolutional Neural Networks (CNNs), the system analyzes image gradients and facial artifacts to distinguish between authentic human captures and synthetic deepfakes.

##  Key Features
* **Facial Artifact Analysis:** Detects inconsistencies in lighting, skin texture, and eye reflections.
* **High-Performance Model:** Powered by a fine-tuned **ResNet-50** architecture for high accuracy.
* **Web-Based Interface:** A user-friendly Flask-based dashboard for instant image verification.
* **Real-time Confidence Scoring:** Provides a percentage-based reliability score for every scan.

##  Tech Stack
* **Backend:** Python, Flask
* **AI/ML:** PyTorch, Torchvision, OpenCV
* **Frontend:** HTML5, CSS3, JavaScript
* **Development:** VS Code, Git
